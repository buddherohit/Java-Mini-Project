🎬 Movie Ticket Booking System

A Java Swing based Movie Ticket Booking System that allows users to book movie tickets, select seats, make payments, and manage bookings through a user-friendly interface.

🚀 Features
🔐 User Login & Registration
🎬 Movie Selection
⏰ Show Timing Selection
🎟️ Seat Selection (Theatre Layout)
🔒 Seat Availability Lock
💳 Payment Simulation
📄 Booking Confirmation
📋 Booking History
👨‍💼 Admin Panel
🎯 Recommended Movies
🎨 Modern Professional UI
🛠️ Technologies Used
Java
Java Swing (GUI)

Event Handling
